ThreatLens — REST API Endpoint Map

4 endpoint groups · Method badges · Auth requirements · Rate limits · Avg response times

🔐 Authentication Group
User login & registration · bcrypt · JWT
POST /api/auth/login
No rate limit ~120ms
Body: { email, password } · Process: bcrypt verify → JWT sign (HS256, 24h)
Returns: { token, user } · Errors: 401 invalid creds · 422 missing fields
POST /api/auth/register
No rate limit ~180ms
Body: { email, password, name } · Process: bcrypt hash (12 rounds) → store user
Returns: { token, user } · Errors: 409 email exists
⚡ Core Analysis Group
Main IOC analysis · Multi-source orchestration
POST /api/ioc-v2
🔐 JWT optional 4/min · 100/day ~3800ms avg
Body: { value: string, type: "ip"|"domain"|"url"|"hash" }
Process: Validate → Sanitize → Cache lookup → 6 parallel APIs → Normalise → Score
Returns: { analysis, risk_score, risk_level, cached, sources[] }
Errors: 400 invalid · 429 rate limit · 500 upstream failure
📊 Dashboard Group
Aggregated statistics · Chart data · Cached 30s
GET /api/dashboard-v2
🔐 JWT required No strict limit ~400ms
Process: MongoDB aggregation pipeline · 30s server-side cache
Returns: { totalScans, riskDistribution, typeBreakdown, recentActivity, topThreats, 10+ chart datasets }
Errors: 401 unauthenticated · 500 DB error
📜 History Group
Paginated history · Filterable · Export CSV/JSON
GET /api/history-v2
🔐 JWT required ~85ms
Query: ?page=1&limit=20&type=ip&search=192.168 · Returns: { items[], total, pages }
DELETE /api/history-v2
🔐 JWT required ~50ms
Clears all history for authenticated userId · Returns: { deleted: count }
GET /api/history-v2/export
🔐 JWT required ~200ms
Query: ?format=csv|json · Returns file download with Content-Disposition header
🌐 Domain Intelligence Group
WHOIS · DNS · SSL certificate data · SSRF-protected
GET /api/domain-intel
🔐 JWT optional 4/min ~1200ms
Query: ?domain=example.com · Fetches: WHOIS registration, DNS records (A/MX/NS), SSL cert info
Security: SSRF-protected via safeFetch() blocklist · Returns: { whois, dns, ssl }
